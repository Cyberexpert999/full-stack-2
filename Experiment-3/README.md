
<h1>Learning Outcomes </h1> 

1. Understand the concept of client-side routing and differentiate it from server-side routing.

2. Install and configure React Router DOM in a React application.

3. Implement basic routing to navigate between multiple pages such as Home, About, Contact, and Products without page reload.

4. Create dynamic routes using URL parameters and access them using the useParams() hook.

5. Implement nested routes and layout structures using Outlet.

6. Perform programmatic navigation using the useNavigate() hook.

7. Implement protected routes using authentication guards and Navigate.

8. Use NavLink and Link components to create navigation with active state highlighting.

9. Work with query parameters using the useSearchParams() hook.
